# DBLP-ACM Publications

## Description
Bibliographic record matching between DBLP and ACM databases

## Files
- DBLP2.csv - DBLP publication records
- ACM.csv - ACM publication records
- DBLP-ACM_perfectMapping.csv - Ground truth matches


## Attributes
id, title, authors, venue, year

## Source
https://dbs.uni-leipzig.de/research/projects/benchmark-datasets-for-entity-resolution
