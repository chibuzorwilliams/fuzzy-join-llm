# Amazon-Google Products

## Description
Product matching between Amazon and Google product listings (software/hardware)

## Files
- Amazon.csv - Amazon product listings (1,363 records)
- GoogleProducts.csv - Google product listings (3,226 records)
- Amazon_GoogleProducts_perfectMapping.csv - Ground truth matches (1,300 pairs)


## Attributes
id, title, description, manufacturer, price

## Source
https://dbs.uni-leipzig.de/research/projects/benchmark-datasets-for-entity-resolution
