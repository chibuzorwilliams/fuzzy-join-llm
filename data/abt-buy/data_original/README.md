# Abt-Buy Electronics

## Description
Product matching between Abt.com and Buy.com (electronics products)

## Files
- Abt.csv - Abt product listings (1,081 records)
- Buy.csv - Buy product listings (1,092 records)
- Abt_Buy_perfectMapping.csv - Ground truth matches (1,097 pairs)


## Attributes
id, name, description, price

## Source
https://dbs.uni-leipzig.de/research/projects/benchmark-datasets-for-entity-resolution
