# DBLP-Scholar Publications

## Description
Bibliographic record matching between DBLP (2,616) and Google Scholar (64,263)

## Files
- DBLP1.csv - DBLP publication records (2,616 records)
- Scholar.csv - Google Scholar records (64,263 records)
- DBLP-Scholar_perfectMapping.csv - Ground truth matches (5,347 pairs)


## Attributes
id, title, authors, venue, year

## Source
https://dbs.uni-leipzig.de/research/projects/benchmark-datasets-for-entity-resolution
